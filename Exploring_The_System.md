# Exploring The System

## ls Commands
- `ls` list the contents of the directory
- `ls usr` list the given directory
- `ls *` list everything 
- `ls -l` long list 

## Notes
Commands follow the following structure `command -options arguments`
`--` means a long option (words)
`-` means a short options

### Types of commands
- executable
- shell builtin
- shell function
- alias
